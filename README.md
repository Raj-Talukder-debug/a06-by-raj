Digitools – AI Marketplace for the Modern Generation

Digitools is a modern website where people can discover and buy the latest AI tools. It is designed for "AI Hunters" who want a fast, clean, and interactive shopping experience.
✨ Features

Modern UI: Features a sleek navbar and an eye-catching banner.

Easy Browsing: Users can easily find tools and see their details.

Shopping Cart: A simple and interactive section to manage selected tools.

🛠️ Tech Stack
React – For building the user interface.

Tailwind CSS – For modern and fast styling.

daisyUI – For beautiful UI components.
